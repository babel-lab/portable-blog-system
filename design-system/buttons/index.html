G-05 Buttons

Phase 2-2-a 定義的 .lab-button 元件，含 5 變體 + 2 size + disabled 狀態。所有按鈕完全使用 Phase 2-1 token。

Variants（變體 × 5）

Primary Secondary Outline Text Disabled

Sizes（size × 3）

以 primary 為例；可套用於任一變體。

Small (sm) Default Large (lg)

Disabled × 變體

.lab-button--disabled modifier 可與任一變體疊加；亦支援原生 :disabled / [disabled]。

Primary Secondary Outline Text

互動測試

  • Hover：滑鼠移至按鈕觀察色彩變化（primary / secondary 加深；outline 反白；text 加底線）
  • Focus：用 Tab 鍵跳到按鈕，應出現 2px primary outline + 2px offset
  • Active：按住 primary / secondary 按鈕，背景再加深一階
  • Disabled：opacity: 0.5 + cursor: not-allowed + pointer-events: none
