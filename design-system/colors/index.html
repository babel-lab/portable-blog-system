G-02 Colors

Phase 2-1 已定義的 --lab-color-* 全部 token，共 20 個。每個色塊以 inline style 直接套用對應 token 變數，便於 DevTools 即時對照。

Semantic Colors（語意色 × 7）

--lab-color-primary
--lab-color-accent
--lab-color-text
--lab-color-muted
--lab-color-background
--lab-color-surface
--lab-color-border

Link Colors（連結色 × 3）

--lab-color-link
--lab-color-link-hover
--lab-color-link-visited

Grayscale（灰階 × 10）

--lab-color-gray-50
--lab-color-gray-100
--lab-color-gray-200
--lab-color-gray-300
--lab-color-gray-400
--lab-color-gray-500

主題切換預覽

切換 <body> 的 lab-site--github ↔ lab-site--blogger class（DevTools Elements 手動編輯測試），--lab-color-primary 色塊應隨主題變色（github 藍 ↔ blogger 棕）。
